FizzBuzz is a program that takes an integer input and gives a String output
if the integer input is divisible by 3, the program prints "fizz"
if the integer input is divisible by 5, the program prints "buzz"
if the integer input is divisible by 3 and 5, the program prints "fizzbuzz"

the first integer input is the number of test cases you would like to run

the main method is in FizzBuzz.java
